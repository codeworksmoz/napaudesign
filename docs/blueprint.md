# **App Name**: Napau Folio

## Core Features:

- Portfolio Showcase: Display a curated collection of 'napau Design & Arte' projects with high-quality images and detailed descriptions.
- Project Filtering & Categorization: Enable users to browse and filter projects by specific categories (e.g., branding, web design, illustration) for easy navigation.
- About & Contact Information: A dedicated section presenting the brand's philosophy, team, services, and an interactive contact form for inquiries.
- Generative Mood Board Tool: An AI-powered tool that assists prospective clients by generating preliminary visual mood boards and design concepts based on their text-based design brief and preferences.
- Admin Content Management: A secure administrative interface for 'napau Design & Arte' to easily upload new projects, update existing content, and manage portfolio details without coding.

## Style Guidelines:

- Primary accent color: A rich, warm gold (#D89926) to evoke luxury and creativity, derived from the brand's logo.
- Background color: A subtle, desaturated off-white (#F9F5EF) providing a clean and elegant canvas that highlights content and contrasts beautifully with the gold accents.
- Accent color: A deep, reddish-brown (#913B14) as an analogous complement to the primary gold, used for high-contrast elements and calls to action.
- Headline and body font: 'Poppins' (sans-serif) for a modern, rounded, and contemporary feel that aligns with the minimalist and artistic branding, using varying weights for clear hierarchy.
- Custom, elegant line-art icons inspired by the organic 'drop' and 'wave' shapes present in the logo, maintaining a minimalist aesthetic.
- Minimalist and spacious layouts featuring ample negative space, clean lines, and centered compositions to create an elegant and focused user experience.
- Subtle and smooth micro-interactions and transitions, potentially including a slight shimmer effect on hover for gold elements to enhance the metallic and brilliant feel.
